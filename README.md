## 部署步骤
1. download源代码
2. 新建mysql数据库apimanager，导入webapp下apimanager.sql
3. 修改jdbc.properties
4. 在eclipse或idea中配置应用服务器，如tomcat；或者直接maven编译打包war，在应用服务器当中启动即可。
